# Sistema de Cadastro de Alunos
 
Projeto em Java para gerenciar alunos pelo console, aplicando os conceitos de CRUD e Programação Orientada a Objetos.
 
---
 
## Funcionalidades
 
- **Cadastrar** aluno com nome, idade, matrícula e curso
- **Listar** todos os alunos cadastrados
- **Editar** dados de um aluno pela matrícula
- **Remover** aluno com confirmação

---
 
## Conceitos aplicados
 
POO, encapsulamento, getters/setters, ArrayList, for-each, try-catch e Scanner.
